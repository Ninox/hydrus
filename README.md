# Hydrus
----------------------------------
A lightweight and fast enough WSGI server for Python.

# Dependences
----------------------------------
**hydrus** is written by C++11, these build tools are required:

+ Python 2.7+
+ Cython
+ GCC 4.7+ / MSVC 2012+

And these 3rd-party library are used in hydrus:

+ http-parser
+ libuv

that's all :smile:
